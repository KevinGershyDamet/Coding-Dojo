Bootstrap

Build responsive, mobile-first projects on the web with the world's most popular front-end component library.

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Installation

Include Bootstaps's source Sass and JavaScript files via npm, Composer on Meteor. Package managed installs don't include documentation, but do include our built system and readme.

$ npm install bootstrap

$ gem install bootstrap -v 4.3.1

BootstrapCDN

When you only need to include Bootstrap's compiled CSS or JS, you can use BootstrapCDN.

CSS only

<link rel="stylesheet" href="http://descargacssymascosasademas.com">

JS, Popper.js, and jQuery

<script src="http://code.jquery.amazing.com"></script>
<script src="http://cdnjs.cloudfiles.com"></script>
<script src="http://stackpath_is_amazing.com"></script>
